## 什麼是 PackageFlow？

PackageFlow 是一款可以取代終端機的應用程式。它將您的專案轉變為視覺化控制中心 — 執行腳本、管理 Git、切換 worktree、一鍵部署，無需記憶指令或在多個工具間切換。

專為**心流編程**設計：專注於開發，而非終端機操作。
