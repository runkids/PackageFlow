## 什么是 PackageFlow？

PackageFlow 是一款可以取代终端的应用程序。它将您的项目转变为可视化控制中心 — 运行脚本、管理 Git、切换 worktree、一键部署，无需记忆命令或在多个工具间切换。

专为**心流编程**设计：专注于开发，而非终端操作。
