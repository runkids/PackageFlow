# 键盘快捷键

PackageFlow 所有键盘快捷键的完整参考。

## 概览

PackageFlow 专为键盘优先的工作流设计。学习这些快捷键以更快导航。

## 快速参考

### 全局快捷键

| 快捷键 | 操作 |
|--------|------|
| <kbd>Cmd</kbd> + <kbd>K</kbd> | 快速切换 worktree |
| <kbd>Cmd</kbd> + <kbd>1</kbd> | 前往项目标签 |
| <kbd>Cmd</kbd> + <kbd>2</kbd> | 前往工作流标签 |
| <kbd>Cmd</kbd> + <kbd>,</kbd> | 打开设置 |
| <kbd>Cmd</kbd> + <kbd>/</kbd> | 显示所有快捷键 |
| <kbd>Cmd</kbd> + <kbd>R</kbd> | 刷新当前视图 |
| <kbd>Cmd</kbd> + <kbd>N</kbd> | 新建项目（依上下文） |
| <kbd>Cmd</kbd> + <kbd>S</kbd> | 保存当前项目 |
| <kbd>Cmd</kbd> + <kbd>W</kbd> | 关闭当前面板 |
| <kbd>Escape</kbd> | 关闭对话框 / 取消 |

### 导航

| 快捷键 | 操作 |
|--------|------|
| <kbd>Cmd</kbd> + <kbd>↑</kbd> | 上一个项目 |
| <kbd>Cmd</kbd> + <kbd>↓</kbd> | 下一个项目 |
| <kbd>Tab</kbd> | 下一个可聚焦元素 |
| <kbd>Shift</kbd> + <kbd>Tab</kbd> | 上一个可聚焦元素 |

### 项目面板

| 快捷键 | 操作 |
|--------|------|
| <kbd>Cmd</kbd> + <kbd>O</kbd> | 打开项目文件夹 |
| <kbd>Cmd</kbd> + <kbd>I</kbd> | 导入项目 |
| <kbd>Cmd</kbd> + <kbd>F</kbd> | 搜索项目 |
| <kbd>Delete</kbd> | 移除选中的项目 |

### Git 面板

| 快捷键 | 操作 |
|--------|------|
| <kbd>Cmd</kbd> + <kbd>Enter</kbd> | 提交暂存的变更 |
| <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd> | 暂存所有变更 |
| <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>U</kbd> | 取消暂存所有变更 |
| <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd> | Push 变更 |
| <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>L</kbd> | Pull 变更 |

### 工作流编辑器

| 快捷键 | 操作 |
|--------|------|
| <kbd>Cmd</kbd> + <kbd>Z</kbd> | 撤销 |
| <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>Z</kbd> | 重做 |
| <kbd>Cmd</kbd> + <kbd>C</kbd> | 复制选中的节点 |
| <kbd>Cmd</kbd> + <kbd>V</kbd> | 粘贴节点 |
| <kbd>Cmd</kbd> + <kbd>A</kbd> | 选择所有节点 |
| <kbd>Delete</kbd> | 删除选中的节点 |
| <kbd>Cmd</kbd> + <kbd>E</kbd> | 运行工作流 |

### 终端

| 快捷键 | 操作 |
|--------|------|
| <kbd>Cmd</kbd> + <kbd>C</kbd> | 复制（选中文本时） |
| <kbd>Cmd</kbd> + <kbd>K</kbd> | 清除终端 |
| <kbd>Ctrl</kbd> + <kbd>C</kbd> | 停止运行中的脚本 |

## 自定义快捷键

### 编辑快捷键

1. 前往**设置** → **键盘快捷键**
2. 找到要更改的快捷键
3. 点击当前的按键组合
4. 按下您的新按键组合
5. 点击**保存**

<!-- TODO: Add screenshot of shortcut editor -->

### 重置为默认值

1. 前往**设置** → **键盘快捷键**
2. 点击**全部重置为默认值**
3. 确认重置

### 冲突检测

PackageFlow 会在以下情况警告您：
- 快捷键与另一个冲突
- 快捷键可能与系统快捷键冲突

## 快捷键分类

### 导航快捷键

在应用程序中快速移动：

- 切换标签
- 导航项目列表
- 打开/关闭面板

### 操作快捷键

执行常见操作：

- 运行脚本
- 提交变更
- 部署项目

### 编辑器快捷键

标准编辑快捷键：

- 撤销/重做
- 复制/粘贴
- 全选

## 高级用户提示

### 逐步学习

1. 从最常用的开始：<kbd>Cmd</kbd> + <kbd>K</kbd>
2. 每天新增一个快捷键
3. 练习直到成为肌肉记忆

### 快速切换最重要

<kbd>Cmd</kbd> + <kbd>K</kbd> 是最强大的快捷键：
- 从任何地方都可使用
- 输入时即时过滤
- 优先显示最近的 worktree

## 无障碍

### 焦点管理

PackageFlow 为键盘导航维护正确的焦点顺序。

### 屏幕阅读器支持

所有交互元素都有适当的标签供屏幕阅读器使用。

### 减少动态效果

启用减少动态效果设置时，快捷键运作方式相同。

## 平台说明

### macOS

所有快捷键使用 <kbd>Cmd</kbd> 作为修饰键。

### Windows/Linux（即将推出）

快捷键将使用 <kbd>Ctrl</kbd> 代替 <kbd>Cmd</kbd>。

## 疑难排解

### 快捷键无效

1. 检查是否有对话框打开（按 <kbd>Escape</kbd>）
2. 在设置中验证快捷键
3. 检查与其他应用程序的冲突
4. 尝试重置为默认值

### 与系统快捷键冲突

某些系统快捷键优先：
- <kbd>Cmd</kbd> + <kbd>Space</kbd>（Spotlight）
- <kbd>Cmd</kbd> + <kbd>Tab</kbd>（应用程序切换器）

PackageFlow 无法覆盖这些。

### 自定义快捷键未保存

1. 检查冲突
2. 确保组合有效
3. 尝试不同的组合
