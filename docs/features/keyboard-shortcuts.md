# Keyboard Shortcuts

Complete reference for all keyboard shortcuts in PackageFlow.

## Overview

PackageFlow is designed for keyboard-first workflows. Learn these shortcuts to navigate faster.

## Quick Reference

### Global Shortcuts

| Shortcut | Action |
|----------|--------|
| <kbd>Cmd</kbd> + <kbd>K</kbd> | Quick switch worktrees |
| <kbd>Cmd</kbd> + <kbd>1</kbd> | Go to Projects tab |
| <kbd>Cmd</kbd> + <kbd>2</kbd> | Go to Workflows tab |
| <kbd>Cmd</kbd> + <kbd>,</kbd> | Open Settings |
| <kbd>Cmd</kbd> + <kbd>/</kbd> | Show all shortcuts |
| <kbd>Cmd</kbd> + <kbd>R</kbd> | Refresh current view |
| <kbd>Cmd</kbd> + <kbd>N</kbd> | New item (context-dependent) |
| <kbd>Cmd</kbd> + <kbd>S</kbd> | Save current item |
| <kbd>Cmd</kbd> + <kbd>W</kbd> | Close current panel |
| <kbd>Escape</kbd> | Close dialog / Cancel |

### Navigation

| Shortcut | Action |
|----------|--------|
| <kbd>Cmd</kbd> + <kbd>↑</kbd> | Previous project |
| <kbd>Cmd</kbd> + <kbd>↓</kbd> | Next project |
| <kbd>Tab</kbd> | Next focusable element |
| <kbd>Shift</kbd> + <kbd>Tab</kbd> | Previous focusable element |

### Project Panel

| Shortcut | Action |
|----------|--------|
| <kbd>Cmd</kbd> + <kbd>O</kbd> | Open project folder |
| <kbd>Cmd</kbd> + <kbd>I</kbd> | Import project |
| <kbd>Cmd</kbd> + <kbd>F</kbd> | Search projects |
| <kbd>Delete</kbd> | Remove selected project |

### Git Panel

| Shortcut | Action |
|----------|--------|
| <kbd>Cmd</kbd> + <kbd>Enter</kbd> | Commit staged changes |
| <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd> | Stage all changes |
| <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>U</kbd> | Unstage all changes |
| <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd> | Push changes |
| <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>L</kbd> | Pull changes |

### Workflow Editor

| Shortcut | Action |
|----------|--------|
| <kbd>Cmd</kbd> + <kbd>Z</kbd> | Undo |
| <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>Z</kbd> | Redo |
| <kbd>Cmd</kbd> + <kbd>C</kbd> | Copy selected nodes |
| <kbd>Cmd</kbd> + <kbd>V</kbd> | Paste nodes |
| <kbd>Cmd</kbd> + <kbd>A</kbd> | Select all nodes |
| <kbd>Delete</kbd> | Delete selected nodes |
| <kbd>Cmd</kbd> + <kbd>E</kbd> | Execute workflow |

### Terminal

| Shortcut | Action |
|----------|--------|
| <kbd>Cmd</kbd> + <kbd>C</kbd> | Copy (when text selected) |
| <kbd>Cmd</kbd> + <kbd>K</kbd> | Clear terminal |
| <kbd>Ctrl</kbd> + <kbd>C</kbd> | Stop running script |

## Customizing Shortcuts

### Edit a Shortcut

1. Go to **Settings** → **Keyboard Shortcuts**
2. Find the shortcut to change
3. Click the current key combination
4. Press your new key combination
5. Click **Save**

<!-- TODO: Add screenshot of shortcut editor -->

### Reset to Defaults

1. Go to **Settings** → **Keyboard Shortcuts**
2. Click **Reset All to Defaults**
3. Confirm the reset

### Conflict Detection

PackageFlow warns you when:
- A shortcut conflicts with another
- A shortcut may conflict with system shortcuts

## Tips for Power Users

### Learn Gradually

1. Start with the most common: <kbd>Cmd</kbd> + <kbd>K</kbd>
2. Add one new shortcut per day
3. Practice until it's muscle memory

## Platform Notes

### macOS

All shortcuts use <kbd>Cmd</kbd> as the modifier key.

### Windows/Linux (Coming Soon)

Shortcuts will use <kbd>Ctrl</kbd> instead of <kbd>Cmd</kbd>.

## Troubleshooting

### Shortcut Not Working

1. Check if a dialog is open (press <kbd>Escape</kbd>)
2. Verify the shortcut in Settings
3. Check for conflicts with other apps
4. Try resetting to defaults

### Conflict with System Shortcuts

Some system shortcuts take precedence:
- <kbd>Cmd</kbd> + <kbd>Space</kbd> (Spotlight)
- <kbd>Cmd</kbd> + <kbd>Tab</kbd> (App switcher)

PackageFlow cannot override these.

### Custom Shortcut Not Saving

1. Check for conflicts
2. Ensure the combination is valid
3. Try a different combination
